# 증명 이미지 폴더

이미지는 **이 폴더**에 넣으면 됩니다.

## 폴더 위치 (복사해서 사용)

```
c:\Users\User\Desktop\mit-challenge\public\images
```

- **파일 탐색기에서 열기**: 위 경로를 복사 → Win+E로 탐색기 열기 → 주소창에 붙여넣기 → Enter
- **Cursor 왼쪽 폴더**: 프로젝트 루트(`mit-challenge`) 열린 상태에서 **public** → **images** 순서로 펼치기

## 필요한 파일

| 파일명 | 용도 |
|--------|------|
| total-account.png | 총계좌 스크린샷 (맨 위 1장) |
| profit-1.png | 수익 증명 1 |
| profit-2.png | 수익 증명 2 |
| profit-3.png | 수익 증명 3 |
| profit-4.png | 수익 증명 4 |

이미 넣어 두었으면 서버 재시작 후 브라우저 **새로고침(F5)** 해보세요.

## 이미지가 안 보일 때

1. 브라우저에서 직접 열어보기: http://localhost:3001/images/total-account.png  
   - 보이면 → 페이지 새로고침  
   - 404면 → 터미널에서 `npm run dev` 실행한 위치가 이 프로젝트(`mit-challenge`)인지 확인
2. Cursor에서 **파일 → 폴더 열기** → `c:\Users\User\Desktop\mit-challenge` 선택 후 다시 실행
