# 개선 방향 (고민 정리)

## 왜 체감이 안 될 수 있나

1. **첫 화면에 할 일이 너무 많음**  
   히어로 → 스탯 → 총계좌 → 캐러셀 → 영상 → CTA까지 한 페이지에 다 있으면, “한 가지 메시지”가 안 잡힌다.

2. **증거와 메시지가 떨어져 있음**  
   “$30,000+”라고 하고, 스탯·총계좌가 아래에 있으면 설득이 한 번에 안 된다. **말 직후에 증거**가 와야 한다.

3. **TWS는 “링크로 검증”을 보여줌**  
   “Lihat Testimoni Member Sepenuhnya Disini”, “Cek History Seluruh Trade”처럼 **볼 수 있다는 링크**가 있으면 신뢰가 올라간다.

4. **스크롤이 평평함**  
   섹션이 그냥 나열되면 단조롭다. 스크롤로 **등장**하는 느낌을 주면 퀄리티가 올라간다.

## 이번에 적용한 개선

| 항목 | 내용 |
|------|------|
| **Hero + 증거 한 덩어리** | 헤드라인·서브 바로 아래에 총계좌 이미지 배치 → “말하고 곧바로 보여주기” |
| **히어로 배경** | 히어로 영역에 은은한 그라데/형태 추가 → 첫 화면이 덜 평평하게 |
| **스크롤 등장** | Stats, Proof, Video, 하단 CTA에 `whileInView`로 통일된 등장 애니메이션 |
| **Lihat bukti lengkap** | 증명 섹션에 “Lihat bukti lengkap” 링크 추가 (URL은 상수로 관리) |
